# 🛡 USB Input Guard

**USB Input Guard** — это настольное приложение для защиты от атак типа **BadUSB** и автоматизированного ввода с USB-устройств.
Система анализирует тайминги нажатий клавиш, выявляет подозрительно быстрый ввод и автоматически блокирует клавиатуру при обнаружении атаки.

Приложение предназначено для Windows и использует мониторинг USB-устройств через WMI.

---

## 🚀 Возможности

- 🔍 Мониторинг подключаемых USB-устройств (VID / PID)
- ⚡ Анализ скорости ввода (тайминг-детектор)
- 🛑 Автоматическая блокировка клавиатуры при обнаружении BadUSB
- 🔐 Защита с подтверждением администратора
- 🟢 Отображение статуса защиты в реальном времени
- 📜 Лог событий в GUI
- 🧪 Встроенный тест скорости печати
- 📋 Управление whitelist (доверенные устройства)
- 🔑 Пароль администратора для критических действий

---

## 🧠 Как работает защита

1. При подключении USB-устройства система проверяет его VID/PID.
2. Если устройство не находится в whitelist — начинается мониторинг ввода.
3. Анализируется минимальный интервал между нажатиями клавиш.
4. Если интервал слишком мал (подозрительно быстрый ввод):
   - показывается предупреждение
   - клавиатура блокируется
   - требуется ввод пароля администратора
5. После отключения устройства защита автоматически восстанавливается.

## 🔧 Настройки

### Минимальный допустимый интервал ввода

По умолчанию:

```python
MIN_ALLOWED_DELTA = 0.005  # секунды
```

Можно изменить через интерфейс:
> Controls → Set typing speed threshold

Рекомендуемый диапазон:
```
0.005 – 0.02 секунд
```

## 🧪 Тест скорости печати

Функция позволяет измерить:
- Средний интервал между нажатиями
- Минимальный интервал
- Рекомендуемый безопасный порог

Доступно в:
> Controls → Test my typing speed

---

## 📋 Whitelist (доверенные устройства)

Для добавления устройства:

1. Нажмите:
   > Add connected device to trusted
2. Подключите устройство
3. Подтвердите добавление в консоли

Данные сохраняются в:

```
usb_whitelist.json
```

Формат:

```json
[
    {
        "vid": "046D",
        "pid": "C31C",
        "name": "USB Keyboard"
    }
]
```

## 🔐 Система безопасности

При обнаружении подозрительного поведения:

- 🔴 Появляется окно SECURITY ALERT
- ⛔ Клавиатура блокируется
- Требуется ввод пароля администратора
- Нельзя закрыть окно без подтверждения

Пароль задаётся в `ui.py`:

```python
ADMIN_PASSWORD = "1234"
```

⚠ Рекомендуется изменить пароль перед использованием.

## 🧩 Основные компоненты

### main.py
Запуск backend + GUI.

### usb_monitor.py
Мониторинг USB через WMI (`Win32_PnPEntity`).

### timing_detector.py
Анализ скорости ввода через `time.perf_counter()`.

### input_blocker.py
Глобальный перехват клавиатуры через `keyboard.hook()`.

### ui.py
Интерфейс на Tkinter.
